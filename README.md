# Cloud-Based Banking System (Python + OOP + Supabase)

A secure, scalable banking simulation showcasing OOP principles and cloud integration.

## Features
- Account types with rules (Savings interest, Checking overdraft)
- Operations: Deposit, withdraw, transfer
- Cloud DB with auth

## Tech Stack
- Python, Supabase, etc. (Will be updated)

## Setup
1. Clone: `git clone <url>`
2. `pip install -r requirements.txt`
3. Add .env with Supabase keys (see below)
